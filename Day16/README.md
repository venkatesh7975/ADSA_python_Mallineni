# Day 16: Advanced Data Structures

## Overview
This folder contains resources, notes, and code implementations for Day 16 of the Data Structures and Algorithms learning journey. 

## Topics Covered
1. **Binary Search Tree (BST)**
   - Implementation using purely recursive functions.
   - Insertion, Searching, and Deletion.
   - Time complexities.
2. **Priority Queues & Heaps**
   - Min-Heaps and Max-Heaps.
   - Easy LeetCode problem solutions using `heapq`.
3. **Hashing**
   - Dictionaries and Sets.
   - Frequency maps.
   - Easy LeetCode problem solutions using hash maps and sets.

## Files
- `Day16_Notes.md`: Comprehensive theoretical notes on the day's topics.
- `Day16_Interview_QA.md`: Common interview questions and answers.
- `bst.py`: Purely recursive BST implementation.
- `minheap_easy_leetcode.py`: Solutions to Min-Heap easy problems.
- `maxheap_easy_leetcode.py`: Solutions to Max-Heap easy problems.
- `hashing_easy_leetcode.py`: Solutions to Hashing easy problems.
