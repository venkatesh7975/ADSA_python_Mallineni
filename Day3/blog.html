Key Concepts

1. Lists & Slicing

Lists are mutable sequences. You can extract subsets using slicing syntax `[start:stop:step]`.

fruits = ["apple", "banana", "cherry", "date"]
subset = fruits[1:3]  # ["banana", "cherry"]
reversed = fruits[::-1]

2. Functions

Functions let you encapsulate code for reuse. They can take parameters and return values.

def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"
    
print(greet("Alice"))       # Hello, Alice!
print(greet("Bob", "Hi"))   # Hi, Bob!

1. How do you create an empty list in Python?

  • A) list = {}
  • B) list = ()
  • C) list = []
  • D) list = ""
View Answer

Answer: C (Square brackets are used to define lists)

2. What is the index of the first element in a list?

  • A) 1
  • B) 0
  • C) -1
  • D) None
View Answer

Answer: B (Python lists are zero-indexed)

3. How do you access the last element of a list `lst`?

  • A) lst[last]
  • B) lst[-1]
  • C) lst[0]
  • D) lst.last()
View Answer

Answer: B (Negative indexing starts from the end of the sequence)

4. What does the slice `lst[1:4]` return?

  • A) Elements at indices 1, 2, 3, 4
  • B) Elements at indices 1, 2, 3
  • C) Elements at indices 0, 1, 2, 3
  • D) Only element 4
View Answer

Answer: B (Slicing goes up to, but does not include, the stop index)

5. What is the difference between `.append()` and `.extend()`?

  • A) They are identical
  • B) `.append()` adds one element, `.extend()` merges another iterable
  • C) `.extend()` adds one element, `.append()` merges another iterable
  • D) `.append()` works on strings, `.extend()` works on lists
View Answer

Answer: B (`.append([1,2])` adds the list as a single element, while `.extend([1,2])` adds 1 and 2 as separate elements)

6. How do you insert "apple" at index 1 in `fruits`?

  • A) fruits[1] = "apple"
  • B) fruits.add(1, "apple")
  • C) fruits.insert(1, "apple")
  • D) fruits.append("apple", 1)
View Answer

Answer: C (The `.insert(index, value)` method shifts elements to the right)

7. How does `.remove("apple")` work?

  • A) Removes all occurrences of "apple"
  • B) Removes the first occurrence of "apple"
  • C) Removes the element at index "apple"
  • D) Throws an error
View Answer

Answer: B (`.remove()` deletes the first matching value, not all of them)

8. What does `.pop()` do if no index is provided?

  • A) Removes the first element
  • B) Removes the last element
  • C) Clears the list
  • D) Throws an error
View Answer

Answer: B (`.pop()` defaults to removing and returning the last element)

9. Which function returns the number of elements in a list?

  • A) size()
  • B) count()
  • C) length()
  • D) len()
View Answer

Answer: D (`len(lst)` returns the total number of items)

10. What is the primary difference between a Tuple and a List?

  • A) Tuples are mutable, Lists are immutable
  • B) Lists are mutable, Tuples are immutable
  • C) Tuples can only store numbers
  • D) There is no difference
View Answer

Answer: B (Once a tuple is created, its elements cannot be changed)

11. Which keyword is used to define a function?

  • A) func
  • B) define
  • C) def
  • D) function
View Answer

Answer: C (`def` is the keyword used to declare a function)

12. What happens if a function does not have a `return` statement?

  • A) It returns 0
  • B) It returns False
  • C) It throws an error
  • D) It returns None
View Answer

Answer: D (Python implicitly returns `None` if no `return` is reached)

13. What is a default argument?

  • A) An argument that cannot be changed
  • B) An argument that assumes a specified value if not provided
  • C) An argument without a name
  • D) An argument that causes an error if omitted
View Answer

Answer: B (e.g., `def greet(name="User"):`)

14. In `def func(a, b=2):`, which is a positional argument?

  • A) a
  • B) b
  • C) Both
  • D) Neither
View Answer

Answer: A (`a` is a required positional argument, while `b` has a default value)

15. What does `*args` allow a function to do?

  • A) Accept unlimited keyword arguments
  • B) Accept unlimited positional arguments
  • C) Return multiple values
  • D) Force type checking
View Answer

Answer: B (`*args` packs extra positional arguments into a tuple)

16. What does `**kwargs` pack arguments into?

  • A) A list
  • B) A tuple
  • C) A dictionary
  • D) A string
View Answer

Answer: C (`**kwargs` captures keyword arguments as key-value pairs in a dictionary)

17. If you define a variable inside a function, can you use it outside?

  • A) Yes
  • B) No, it has local scope
  • C) Yes, if it is a list
  • D) Only in a while loop
View Answer

Answer: B (Local variables are destroyed when the function exits)

18. How do you modify a global variable inside a function?

  • A) You can't
  • B) Just assign a new value to it
  • C) Use the `global` keyword
  • D) Pass it as an argument
View Answer

Answer: C (e.g., `global x` before assigning `x = 5`)

19. What does the string method `.replace("a", "b")` do?

  • A) Modifies the original string
  • B) Returns a new string with all "a"s replaced by "b"s
  • C) Replaces only the first "a"
  • D) Throws an error
View Answer

Answer: B (Strings are immutable, so it returns a modified copy)

20. Are strings mutable in Python?

  • A) Yes
  • B) No
  • C) Only if they are inside a list
  • D) Only if defined with triple quotes
View Answer

Answer: B (You cannot change a character in a string via indexing. You must create a new string)
