# Day 11 Quiz: Queues

## 1. What does FIFO stand for?
A) First In First Out
B) Fast In Fast Out
C) First In Fast Out
D) Fast In First Out

**Answer:** A

## 2. In a queue, insertion is done at:
A) Front
B) Rear
C) Middle
D) Anywhere

**Answer:** B

## 3. Which data structure is used for Breadth First Search (BFS)?
A) Stack
B) Queue
C) Tree
D) Graph

**Answer:** B

## 4. In Python, which collection provides O(1) time complexity for both append and pop operations from both ends?
A) list
B) set
C) tuple
D) collections.deque

**Answer:** D

## 5. What is the time complexity of a dequeue operation in an array-based simple queue (like Python list's `pop(0)`)?
A) O(1)
B) O(N)
C) O(log N)
D) O(N^2)

**Answer:** B
