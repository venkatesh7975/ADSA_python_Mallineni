# Day 11: Introduction to Queues

Welcome to Day 11! Today is all about understanding the **Queue**, a fundamental data structure based on the First-In, First-Out (FIFO) principle.

## 🎯 Today's Topics

1. **The FIFO Principle**
   - First In, First Out mechanics.
2. **Core Operations**
   - Enqueue, Dequeue, Peek, and checking for Empty.
3. **Implementation in Python**
   - Leveraging Python's `collections.deque` to achieve O(1) time complexity.
4. **Powerful Applications**
   - Task Scheduling, Print Spoolers, and Breadth-First Search (BFS).
