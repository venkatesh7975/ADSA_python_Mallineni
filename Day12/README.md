# Day 12: Circular Queues and Deques

Welcome to Day 12! Today we advance our queue knowledge by focusing on two specialized and highly useful variants of Queues: **Circular Queues** and **Deques (Double Ended Queues)**.

## 🎯 Today's Topics

1. **Circular Queues (Ring Buffers)**
   - Overcoming the limitations of linear array-based queues.
   - Implementing wrap-around logic using modulo arithmetic.
2. **Deques (Double Ended Queues)**
   - Understanding insertion and deletion at both ends.
   - Using `collections.deque` in Python for O(1) performance.
3. **Advanced Applications**
   - Sliding Window algorithms.
   - Palindrome verification.
   - Memory management and task scheduling.
