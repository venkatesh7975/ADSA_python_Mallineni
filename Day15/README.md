# Day 15: Hashing, Sets, Frequency Maps & BST Mastery

Welcome to Day 15! We are focusing on optimizing code from $O(N^2)$ brute-force solutions down to lightning-fast $O(N)$ or $O(\log N)$ algorithms by using Hash Maps, Sets, and leveraging Binary Search Trees.

## 🎯 Today's Focus
- **Hashing**: How $O(1)$ lookup times change the game for arrays and strings.
- **Sets**: Removing duplicates and rapid membership testing.
- **Frequency Maps**: Using Python's `collections.Counter` and `defaultdict` to solve anagram, permutation, and counting problems.
- **BST Problem Solving**: Using Hash maps alongside BST traversals.
