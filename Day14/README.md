# Day 14: Introduction to Trees & Binary Search Trees (BST)

Welcome to Day 14! Today we graduate from linear data structures (like arrays, stacks, and linked lists) into the beautiful, hierarchical world of **Trees**.

## 🎯 Today's Focus
Trees form the backbone of advanced computer science architectures, from your operating system's file directories to the routing tables in network switches.

We will focus specifically on **Binary Trees** and **Binary Search Trees (BSTs)**, learning how to traverse them and manipulate their properties in $O(\log N)$ time.
