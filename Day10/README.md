# Day 10: Introduction to Linked Lists

Welcome to Day 10! We are leaving behind contiguous arrays and stepping into the dynamic world of node-based data structures. Today is all about understanding the anatomy of a **Singly Linked List**.

## 🎯 Today's Topics

1. **Linked Lists vs Arrays**
   - Memory differences and Time Complexity trade-offs.
2. **The Node Structure**
   - Object-Oriented Programming (OOP) in Python to build nodes.
3. **Traversal**
   - "Walking" the list from head to tail using a while loop.
4. **Basic Insertions**
   - The O(1) magic of inserting at the Head.
   - Traversing to insert at the Tail.
