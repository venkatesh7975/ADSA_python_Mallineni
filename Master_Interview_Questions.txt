100 Python Interview Questions and Answers: Day 1 Topics
(Fundamentals, Variables, Data Types, and Operators)

=========================================
SECTION 1: PYTHON FUNDAMENTALS (Q1 - Q20)
=========================================

1. What is Python?
Answer: Python is a high-level, interpreted, dynamically typed, and general-purpose programming language known for its readability and simple syntax.

2. Who created Python and when was it first released?
Answer: Guido van Rossum created Python, and it was first released in 1991.

3. Why is Python called an "interpreted" language?
Answer: Python code is executed line-by-line by an interpreter rather than being compiled into machine code all at once before execution.

4. What does "dynamically typed" mean in Python?
Answer: It means you don't have to declare the type of a variable when you create one. The Python interpreter infers the type automatically at runtime based on the value assigned.

5. What is PEP 8?
Answer: PEP 8 is the official style guide for Python code, providing guidelines on how to format code (e.g., indentation, naming conventions) for maximum readability.

6. Why is indentation important in Python?
Answer: Unlike languages that use curly braces {} to define code blocks, Python uses indentation (whitespace) to determine the scope of loops, functions, and conditional statements. Incorrect indentation causes an IndentationError.

7. How do you write a comment in Python?
Answer: Single-line comments start with the hash symbol (#). Multi-line comments can be created using triple quotes (''' or """), though technically they are multi-line strings.

8. Is Python case-sensitive?
Answer: Yes. Variables named 'Data' and 'data' are treated as two distinct variables.

9. What are the key features of Python?
Answer: Key features include being easy to read, interpreted, object-oriented, dynamically typed, having a vast standard library, and being cross-platform.

10. What is the difference between Python 2 and Python 3?
Answer: Python 3 is the current, modern version of the language (e.g., print() is a function). Python 2 is legacy and no longer supported.

11. How do you print something to the console in Python?
Answer: Using the built-in print() function. Example: print("Hello World")

12. What does high-level language mean?
Answer: It means the language abstracts away complex machine details like memory management and CPU registers, making it closer to human language.

13. What is a Python script?
Answer: A file containing Python code (usually with a .py extension) intended to be executed from top to bottom.

14. How do you execute a Python script from the command line?
Answer: By typing `python filename.py` or `python3 filename.py`.

15. What are docstrings in Python?
Answer: Docstrings are string literals appearing right after the definition of a function, method, class, or module, used for documentation.

16. What is the purpose of the pass statement?
Answer: 'pass' is a null operation. It is used as a placeholder in empty code blocks where syntax requires a statement, but no action is needed.

17. Can Python be used for frontend web development?
Answer: Primarily, Python is a backend language. While tools like Brython exist, HTML/CSS/JavaScript are standard for frontend. Python excels in backend (Django, Flask).

18. What is CPython?
Answer: CPython is the default, standard, and most widely used implementation of Python, written in C.

19. Why is Python heavily used in Data Science?
Answer: Because of its simple syntax and incredibly powerful libraries like Pandas, NumPy, Matplotlib, and Scikit-learn.

20. What is an IDE? Name two popular ones for Python.
Answer: An Integrated Development Environment (IDE) provides facilities for software development. Popular ones for Python include PyCharm and VS Code.

SECTION 2: VARIABLES (Q21 - Q40)
=========================================

21. What is a variable in Python?
Answer: A variable is a named memory location used to store data. It acts as a reference or a pointer to an object.

22. How do you declare a variable in Python?
Answer: You declare a variable by simply assigning a value to it using the equals sign (=). Example: x = 10.

23. Can a variable name start with a number?
Answer: No. Variable names must start with a letter or an underscore. '1st_place' is invalid; 'place_1' is valid.

24. What characters are allowed in a Python variable name?
Answer: Letters (a-z, A-Z), numbers (0-9), and underscores (_).

25. Are 'MyVar' and 'myvar' the same variable?
Answer: No, Python is case-sensitive, so they are two different variables.

26. Can you use reserved words (keywords) as variable names?
Answer: No. Words like 'if', 'for', 'class', 'while' are reserved by Python and will cause a SyntaxError if used as variable names.

27. What is snake_case?
Answer: A naming convention where words are separated by underscores, widely used for Python variables and functions (e.g., total_sales_amount).

28. What is camelCase?
Answer: A naming convention where the first letter is lowercase and subsequent words start with uppercase (e.g., totalSalesAmount). Snake_case is preferred in Python.

29. How can you assign the same value to multiple variables at once?
Answer: x = y = z = 100.

30. How can you assign different values to multiple variables in a single line?
Answer: name, age, city = "Alice", 25, "New York"

31. Can you change the type of a variable after creating it?
Answer: Yes. Because Python is dynamically typed, you can do: x = 5 (int) followed by x = "Hello" (string).

32. What happens if you try to use a variable before assigning a value to it?
Answer: Python will throw a NameError.

33. How do you delete a variable in Python?
Answer: Using the 'del' keyword. Example: del my_var.

34. What is a global variable?
Answer: A variable declared outside of any function, which can be accessed from anywhere in the script.

35. What is a local variable?
Answer: A variable declared inside a function. It can only be accessed within that specific function.

36. How do you use a global variable inside a function?
Answer: By using the 'global' keyword before the variable name inside the function.

37. Is there a constant variable type in Python?
Answer: Technically no, but by convention, programmers use ALL_CAPS to indicate a variable should be treated as a constant (e.g., PI = 3.14159).

38. What does variable swapping look like in Python without a temporary variable?
Answer: a, b = b, a

39. What does the id() function do?
Answer: It returns the unique memory address (identity) of a variable/object.

40. If a = 10 and b = 10, are they the same object in memory?
Answer: Yes, for small integers, Python optimizes memory by pointing them to the same object (you can verify with id(a) == id(b)).

SECTION 3: DATA TYPES (Q41 - Q70)
=========================================

41. What are the core numeric data types in Python?
Answer: int (integers), float (decimal numbers), and complex (complex numbers).

42. How do you find the data type of a variable?
Answer: Use the built-in type() function. Example: type(my_var).

43. What is a string (str)?
Answer: A string is a sequence of characters enclosed in single, double, or triple quotes.

44. Are strings mutable or immutable?
Answer: Strings are immutable. Once created, you cannot change individual characters in a string object.

45. How do you concatenate two strings?
Answer: Using the + operator. Example: "Hello" + " " + "World".

46. What does repeating a string look like?
Answer: Use the * operator. Example: "Hi" * 3 results in "HiHiHi".

47. What is a Boolean (bool) data type?
Answer: A Boolean represents one of two values: True or False.

48. Is 'True' the same as 'true' in Python?
Answer: No. Booleans in Python must be capitalized: True and False.

49. What is a list in Python?
Answer: A list is an ordered, mutable collection of items enclosed in square brackets []. Example: [1, 2, 3].

50. Can a list hold mixed data types?
Answer: Yes. A list can hold integers, strings, floats, and even other lists. Example: [1, "apple", 3.14].

51. What is a tuple?
Answer: A tuple is an ordered, immutable collection of items enclosed in parentheses (). Example: (1, 2, 3).

52. When should you use a tuple instead of a list?
Answer: Use a tuple when the sequence of data should not be changed (read-only), such as geographic coordinates or days of the week.

53. What is a dictionary (dict)?
Answer: An unordered collection of key-value pairs enclosed in curly braces {}. Example: {"name": "Alice", "age": 25}.

54. Are dictionary keys mutable or immutable?
Answer: Keys must be immutable types (like strings, numbers, or tuples). Lists cannot be used as dictionary keys.

55. What is a set?
Answer: An unordered collection of unique items enclosed in curly braces {}. Duplicates are automatically removed.

56. Can you access elements in a set using an index (e.g., my_set[0])?
Answer: No, sets are unordered, meaning they do not support indexing or slicing.

57. How do you convert a string to an integer?
Answer: Using the int() function. Example: int("10").

58. What happens if you run int("10.5")?
Answer: It raises a ValueError. You must first convert to float: int(float("10.5")).

59. How do you convert an integer to a float?
Answer: Using the float() function. Example: float(5) becomes 5.0.

60. How do you convert a number to a string?
Answer: Using the str() function. Example: str(100) becomes "100".

61. What is type casting?
Answer: The process of explicitly converting a variable from one data type to another.

62. What is implicit type conversion?
Answer: When Python automatically converts one data type to another without programmer intervention (e.g., adding an int and a float results in a float).

63. What evaluates to False in Python?
Answer: False, None, zero (0, 0.0), empty sequences ("", [], ()), and empty mapping ({}).

64. What is the NoneType?
Answer: None is a special constant in Python representing the absence of a value or a null value. Its type is NoneType.

65. What is the difference between a list and a set?
Answer: A list is ordered, allows duplicates, and supports indexing. A set is unordered, does not allow duplicates, and does not support indexing.

66. How do you find the length of a string, list, or dictionary?
Answer: Using the built-in len() function.

67. What is string slicing?
Answer: Accessing a subset of a string using indices. Example: "Python"[0:2] returns "Py".

68. What data type is the result of 10 / 2?
Answer: Float (5.0). Standard division in Python 3 always returns a float.

69. Can you modify an item in a dictionary?
Answer: Yes, dictionaries are mutable. Example: my_dict["age"] = 26.

70. Are floats 100% accurate in Python?
Answer: No, floating-point arithmetic can have minor precision issues due to how computers represent decimals in binary (e.g., 0.1 + 0.2 might equal 0.30000000000000004).

SECTION 4: OPERATORS (Q71 - Q100)
=========================================

71. What are Arithmetic operators?
Answer: Operators used to perform mathematical operations (+, -, *, /, //, %, **).

72. What is the difference between / and // ?
Answer: / performs standard division and returns a float. // performs integer (floor) division and discards the fractional part, returning an integer.

73. What does the modulo operator (%) do?
Answer: It returns the remainder of a division. Example: 10 % 3 returns 1.

74. How do you calculate powers in Python?
Answer: Using the exponentiation operator (**). Example: 2 ** 3 equals 8.

75. What are Comparison (Relational) operators?
Answer: Operators used to compare values (==, !=, >, <, >=, <=). They always return a Boolean.

76. What is the difference between = and == ?
Answer: = is the assignment operator (assigns a value to a variable). == is the equality comparison operator (checks if two values are equal).

77. What does != mean?
Answer: Not equal to. Returns True if operands are different.

78. What are Logical operators?
Answer: Operators used to combine conditional statements (and, or, not).

79. How does the 'and' operator work?
Answer: Returns True only if BOTH operands (conditions) are True.

80. How does the 'or' operator work?
Answer: Returns True if AT LEAST ONE operand (condition) is True.

81. What does the 'not' operator do?
Answer: It inverts the Boolean value. not True becomes False.

82. What is short-circuit evaluation?
Answer: When Python evaluates logical expressions, if the first condition determines the outcome, it stops evaluating. (e.g., in `A or B`, if A is True, B is not evaluated).

83. What are Assignment operators?
Answer: Operators used to assign values to variables (=, +=, -=, *=, /=, %=, etc.).

84. What does x += 5 mean?
Answer: It is shorthand for x = x + 5 (Add and assign).

85. Does Python support ++ or -- increment/decrement operators like C++ or Java?
Answer: No. In Python, you must use += 1 or -= 1.

86. What are Membership operators?
Answer: 'in' and 'not in'. They check if a sequence (like a string, list, or tuple) is present within an object.

87. Give an example of using the 'in' operator.
Answer: "apple" in ["apple", "banana"] returns True.

88. What are Identity operators?
Answer: 'is' and 'is not'. They compare the memory locations of two objects, not just their values.

89. What is the difference between '==' and 'is'?
Answer: '==' checks if values are equal. 'is' checks if the two variables point to the exact same object in memory.

90. If a = [1, 2] and b = [1, 2], what is a == b and what is a is b?
Answer: a == b is True (values are equal). a is b is False (they are two distinct lists in memory).

91. What is the order of precedence for operators in Python (PEMDAS)?
Answer: Parentheses, Exponentiation, Multiplication/Division/Modulo, Addition/Subtraction.

92. How do you override the default operator precedence?
Answer: By using parentheses ().

93. What is the result of 10 > 5 and 5 > 2?
Answer: True. Both conditions are True, so the 'and' statement evaluates to True.

94. What is the result of "Python" > "Java"?
Answer: True. String comparison is alphabetical (lexicographical). 'P' comes after 'J' in the alphabet, so it is considered greater.

95. Can you use + to add a string and an integer?
Answer: No, it raises a TypeError. You must convert the integer to a string first (e.g., "Age " + str(25)).

96. What does True + True evaluate to in Python?
Answer: 2. In arithmetic operations, True acts as 1 and False acts as 0.

97. What is the result of 10 % 2 == 0?
Answer: True. 10 % 2 evaluates to 0, and 0 == 0 is True. This is how you check if a number is even.

98. Can you chain comparison operators in Python?
Answer: Yes. Example: 0 < x < 10 is perfectly valid and checks if x is between 0 and 10.

99. What does the bitwise AND operator (&) do?
Answer: It performs a bit-by-bit logical AND operation on integers. (Generally more advanced than Day 1, but good to know it exists distinct from 'and').

100. If x = 5, what does x *= 3 evaluate to?
Answer: x becomes 15.

SECTION 1: CONDITIONAL STATEMENTS (Q1 - Q25)
=========================================

1. What is an if statement in Python?
Answer: An if statement checks a condition and executes a block of code only if the condition evaluates to True.

2. What does elif mean?
Answer: `elif` stands for "else if" and lets you check additional conditions after the first `if` condition is False.

3. What is the purpose of else?
Answer: `else` executes a block of code when all preceding `if` and `elif` conditions are False.

4. How do you write a simple if-else statement?
Answer: 
```python
if x > 0:
    print("Positive")
else:
    print("Zero or negative")
```

5. Can you have multiple elif blocks?
Answer: Yes. You can use multiple `elif` blocks to test more than two conditions.

6. What happens if you omit the else block?
Answer: The code simply skips the else part when no condition matches; the program continues with the next statement.

7. What is the indentation rule for if statements?
Answer: The body of the `if`, `elif`, and `else` blocks must be indented consistently, usually by 4 spaces.

8. Can you use comparison operators in an if condition?
Answer: Yes. Common comparison operators are `==`, `!=`, `>`, `<`, `>=`, and `<=`.

9. What is a boolean expression?
Answer: A boolean expression evaluates to either `True` or `False`, and it is used in if conditions and loops.

10. How would you check if a variable is equal to 10?
Answer: `if x == 10:`

11. What is the difference between `=` and `==`?
Answer: `=` assigns a value to a variable, while `==` compares two values for equality.

12. How do you combine conditions in Python?
Answer: Use logical operators `and`, `or`, and `not`.

13. What does `and` do?
Answer: `and` returns True only if both operands are True.

14. What does `or` do?
Answer: `or` returns True if at least one of the operands is True.

15. What does `not` do?
Answer: `not` inverts the boolean value, making True into False and vice versa.

16. What is a nested if statement?
Answer: A nested if statement is an `if` statement inside another `if` or `else` block.

17. Why use nested if statements?
Answer: Nested ifs are useful when you need multi-level decision logic and one condition depends on another.

18. What is an example of nested if else?
Answer: 
```python
if x > 0:
    if x % 2 == 0:
        print("Positive even")
    else:
        print("Positive odd")
else:
    print("Zero or negative")
```

19. What is the else-if ladder?
Answer: It is a series of `if` and `elif` conditions where only one matching block runs.

20. How do you avoid deep nesting?
Answer: Use `elif` for sequential checks, functions to break logic into smaller units, and boolean expressions when possible.

21. What is a common use case for if statements in data validation?
Answer: Checking if user input is valid, such as `if age >= 18:` to verify eligibility.

22. What happens if a condition is always True?
Answer: The if block always runs and any following `elif` or `else` blocks are skipped.

23. What happens if a condition is always False?
Answer: The program moves to the next `elif` or `else` block.

24. Can `if` be used without `else`?
Answer: Yes. The `if` block can stand alone with no `elif` or `else`.

25. What is a truthy value in Python?
Answer: A truthy value is any value that evaluates to True in a boolean context, for example non-zero numbers or non-empty strings.

SECTION 2: FOR LOOPS (Q26 - Q50)
=========================================

26. What is a for loop in Python?
Answer: A `for` loop repeats a block of code for each item in a sequence or iterable.

27. How do you write a basic for loop?
Answer: 
```python
for i in range(5):
    print(i)
```

28. What does `range(5)` produce?
Answer: It produces the sequence 0, 1, 2, 3, 4.

29. Can range start from a number other than 0?
Answer: Yes. `range(2, 6)` produces 2, 3, 4, 5.

30. What does `range(1, 10, 2)` do?
Answer: It produces odd numbers from 1 to 9 by stepping 2 each time.

31. How do you loop over a string?
Answer: 
```python
for char in "hello":
    print(char)
```

32. How do you loop over a list?
Answer: 
```python
for item in [1, 2, 3]:
    print(item)
```

33. How do you loop over a dictionary?
Answer: Use `.keys()`, `.values()`, or `.items()`.
Example:
```python
for key, value in my_dict.items():
    print(key, value)
```

34. What is a loop variable?
Answer: The variable that takes the value of each item in the sequence during iteration.

35. What is a common use of for loops?
Answer: Repeating calculations, building lists, printing patterns, or processing items in a collection.

36. What does `for i in range(1, 6):` do?
Answer: It loops over the values 1, 2, 3, 4, 5.

37. How do you create a multiplication table with a for loop?
Answer: Use nested loops or a single loop with arithmetic.
Example:
```python
for i in range(1, 6):
    print(f"2 x {i} = {2 * i}")
```

38. What is string repetition in Python?
Answer: Using the `*` operator with a string and integer, e.g. `"*" * 5` produces `*****`.

39. Can you break out of a for loop early?
Answer: Yes, using the `break` statement.

40. What does `break` do?
Answer: It immediately exits the loop.

41. What does `continue` do?
Answer: It skips the rest of the current iteration and moves to the next one.

42. How do you skip the first item in a loop?
Answer: Start the range at 1 or use an index check with `continue`.

43. What is the output of `for i in range(3): print(i)`?
Answer: 0, 1, 2 on separate lines.

44. How can you loop with custom steps?
Answer: Use the third `step` parameter in `range(start, stop, step)`.

45. Can a for loop iterate over a tuple?
Answer: Yes, tuples are iterable.

46. What happens when the iterable is empty?
Answer: The loop body does not execute.

47. How do you print each character in a name on a new line?
Answer: 
```python
for ch in name:
    print(ch)
```

48. Can you loop over file lines with a for loop?
Answer: Yes. `for line in open('file.txt'):` reads each line from the file.

49. How do you use enumerate in a for loop?
Answer: `for index, item in enumerate(my_list):` gives both index and value.

50. Why use `enumerate()`?
Answer: To access the position of each element while iterating.

SECTION 3: WHILE LOOPS (Q51 - Q70)
=========================================

51. What is a while loop?
Answer: A `while` loop runs as long as a condition remains True.

52. How do you write a simple while loop?
Answer: 
```python
count = 1
while count <= 5:
    print(count)
    count += 1
```

53. What is the loop condition?
Answer: The boolean expression evaluated before each iteration to determine if the loop should continue.

54. How do you avoid an infinite while loop?
Answer: Ensure the loop condition changes and can become False.

55. What is an example of a countdown using while?
Answer: 
```python
n = 5
while n > 0:
    print(n)
    n -= 1
```

56. Can a while loop run zero times?
Answer: Yes, if the condition is False at the start.

57. What is common use for while loops?
Answer: Repeating until a condition is satisfied, like user input validation or countdowns.

58. How do you update the loop variable?
Answer: Inside the loop body, typically with `+=`, `-=`, or another assignment.

59. What is an infinite loop?
Answer: A loop that never ends because its condition never becomes False.

60. How do you safely stop a while loop manually?
Answer: Use `break` inside the loop.

61. Can you use else with while loops?
Answer: Yes, the `else` block executes when the loop ends naturally (not via `break`).

62. What is a common error with while loops?
Answer: Forgetting to update the loop variable or using the wrong condition.

63. How do you print even numbers using while?
Answer: 
```python
n = 2
while n <= 10:
    print(n)
    n += 2
```

64. How do you use input in a while loop?
Answer: Ask for input inside the loop until a valid value is provided.
Example:
```python
while True:
    text = input("Enter yes or no: ")
    if text in ["yes", "no"]:
        break
```

65. Can while loops use `range()`?
Answer: Not directly. `range()` is used with `for` loops, but you can simulate it with an index variable.

66. What should you watch for when using while loops with floating point numbers?
Answer: Precision issues may prevent the condition from becoming False as expected.

67. How do you count from 100 down to 0 by 5 using while?
Answer: 
```python
num = 100
while num >= 0:
    print(num)
    num -= 5
```

68. Why is `while True:` useful?
Answer: It creates a loop that continues until a `break` is encountered, useful for repeated input or event loops.

69. Can a while loop contain another while loop?
Answer: Yes, nested while loops are valid but should be used carefully.

70. When is a while loop better than a for loop?
Answer: When the number of iterations is unknown and depends on changing conditions.

SECTION 4: NESTED LOOPS (Q71 - Q85)
=========================================

71. What is a nested loop?
Answer: A loop placed inside another loop.

72. How does nested loop execution work?
Answer: The inner loop completes all iterations for each single iteration of the outer loop.

73. What is an example of a nested loop?
Answer: 
```python
for i in range(3):
    for j in range(3):
        print(i, j)
```

74. What is a common use of nested loops?
Answer: Working with 2D data like tables, grids, or generating patterns.

75. How would you print a multiplication table using nested loops?
Answer: 
```python
for i in range(1, 6):
    for j in range(1, 6):
        print(i * j, end=" ")
    print()
```

76. Are nested loops only used with for loops?
Answer: No, you can nest `for` loops inside `while` loops and vice versa.

77. What is the performance cost of nested loops?
Answer: It typically increases execution time exponentially, especially when both loops iterate many times.

78. How do you format output inside nested loops?
Answer: Use `end=" "` to print on the same line and `print()` after the inner loop to start a new line.

79. What pattern does this code print?
```python
for i in range(1, 4):
    for j in range(i):
        print("*", end="")
    print()
```
Answer: A right-angle triangle of stars:
*
**
***

80. What is a star pattern?
Answer: A visual shape printed on the console using repeated characters and loops.

81. How do you print a square of stars?
Answer: 
```python
for i in range(4):
    print("* " * 4)
```

82. How do you print a triangle of stars?
Answer: 
```python
for i in range(1, 5):
    print("* " * i)
```

83. What does nested loop control depend on?
Answer: Both the outer and inner loop conditions.

84. How do you avoid deep nested loops when possible?
Answer: Use helper functions, break data into smaller pieces, or use list comprehensions for some cases.

85. Why should you be careful with nested loops in large datasets?
Answer: Because the runtime can grow quickly and become slow or inefficient.

SECTION 5: PRACTICAL PROBLEMS AND DEBUGGING (Q86 - Q100)
=========================================

86. What is FizzBuzz?
Answer: A programming exercise that prints `Fizz` for multiples of 3, `Buzz` for multiples of 5, `FizzBuzz` for multiples of both, and the number otherwise.

87. How do you write FizzBuzz in Python?
Answer: 
```python
for num in range(1, 16):
    if num % 3 == 0 and num % 5 == 0:
        print("FizzBuzz")
    elif num % 3 == 0:
        print("Fizz")
    elif num % 5 == 0:
        print("Buzz")
    else:
        print(num)
```

88. What is dry run?
Answer: Dry run means tracing code by hand to follow variable values and program flow without executing it.

89. Why is dry run useful?
Answer: It helps catch logic errors, understand loops, and verify conditions before running code.

90. What is debugging?
Answer: Debugging is the process of finding and fixing errors or bugs in code.

91. What is an indentation error?

92. What is a common mistake in nested if statements?

93. How do you test a loop with a sample value?

94. What does `print(i, j)` do inside nested loops?

95. How do you print only odd numbers using a for loop?

97. What does `if number % 2 == 0:` check?

98. What value does `range(5, 0, -1)` generate?

99. How do you check whether a string contains a specific character?

100. How can you use loops to build a list of values?
Answer: Initialize an empty list, append values inside the loop, and then use the list after the loop.
Example:

Answer: Yes, `@functools.lru_cache` or `@functools.cache` can automatically memoize a function.

100. If you do a dry run of `fib(3)`, how many frames are pushed to the stack in total?
Answer: `fib(3)` calls `fib(2)` and `fib(1)`. `fib(2)` calls `fib(1)` and `fib(0)`. Total calls including the first is 5 frames pushed to the stack overall.

Day 5 Interview Questions

1. What is the difference between `.append()` and `.extend()` in Python lists?
Answer: `.append()` adds a single element to the end of the list. If you append a list, the entire list is added as one element. `.extend()` iterates over its argument and adds each element to the list, effectively merging lists.

2. Explain list slicing. How do you reverse a list using slicing?
Answer: List slicing extracts a portion of a list using the syntax `list[start:stop:step]`. To reverse a list, we can use `list[::-1]`, which steps backward through the list from end to beginning.

3. Are strings mutable in Python? What about lists?
Answer: Strings are immutable, meaning they cannot be changed after creation. Any string operation that modifies a string actually returns a new string. Lists are mutable; they can be changed in place (e.g., changing an element by index, appending, or removing elements).

4. Write code or explain how you would find the frequency of elements in a list.
Answer: We can iterate through the list and use a dictionary to keep track of counts. 
```python
freq = {}
for item in lst:
    freq[item] = freq.get(item, 0) + 1
```

5. How do you generate all unique pairs from an array of numbers?
Answer: Use two nested loops. The outer loop runs from `i = 0` to `len(arr)-1`, and the inner loop runs from `j = i + 1` to `len(arr)-1`. This ensures each pair `(arr[i], arr[j])` is generated exactly once.
