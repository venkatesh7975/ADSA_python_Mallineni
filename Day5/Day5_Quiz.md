# Day 5 Quiz

1. What is the output of `"Hello"[1:4]`?
   - a) `ell`
   - b) `Hel`
   - c) `ello`
   - d) `llo`

2. Which list method is used to add multiple elements from an iterable to the end of the list?
   - a) `.append()`
   - b) `.insert()`
   - c) `.extend()`
   - d) `.add()`

3. How would you read a space-separated string of numbers into a Python list of integers?
   - a) `list(input())`
   - b) `input().split()`
   - c) `list(map(int, input().split()))`
   - d) `int(input().split())`

4. Given `nums = [10, 20, 30, 40]`, what is `nums[::-1]`?
   - a) `[10, 20, 30, 40]`
   - b) `[40, 30, 20, 10]`
   - c) `[10, 30]`
   - d) `[40, 20]`

5. If you want to count the occurrences of a specific item in a list, which method do you use?
   - a) `.find(item)`
   - b) `.count(item)`
   - c) `.index(item)`
   - d) `.freq(item)`
