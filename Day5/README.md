# Day 5: Strings, Lists, and Problem Solving

## Overview
Today focuses on essential data structure operations using strings and lists. We cover slicing, indexing, built-in methods, and how to read standard input efficiently.

## Topics Covered
- String and List Slicing
- Positive and Negative Indexing
- Common List Methods (`append`, `extend`, `pop`, `remove`, `count`, etc.)
- Iterating and reading various inputs
- Problem solving: Frequencies, sums, products, and nested loops for pairs
