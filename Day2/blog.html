Key Concepts

1. Conditionals

Use `if/elif/else` to execute different blocks of code based on conditions.

age = 18
if age >= 18:
    print("You can vote!")
else:
    print("Too young to vote.")

2. Loops

Loops allow you to repeat a block of code multiple times. `for` loops are great for counting, while `while` loops run until a condition is met.

# For Loop
for i in range(5):
    print("Iteration:", i)

# While Loop
count = 3
while count > 0:
    print(count)
    count -= 1

Test your knowledge on Control Flow and Loops!

1. Which keyword is used to start a conditional statement?

  • A) switch
  • B) when
  • C) if
  • D) check
View Answer

Answer: C (`if` is the starting keyword for conditionals)

2. What does `elif` stand for?

  • A) Else If
  • B) End If
  • C) Evaluate If
  • D) Error If
View Answer

Answer: A (`elif` is Python's shorthand for "else if")

3. How does Python define code blocks (e.g., inside an `if` statement)?

  • A) Curly braces {}
  • B) Indentation (spaces/tabs)
  • C) BEGIN and END keywords
  • D) Parentheses ()
View Answer

Answer: B (Python relies strictly on indentation to define scope)

4. What does a `while` loop do?

  • A) Loops exactly 10 times
  • B) Loops over a list
  • C) Executes a block of code as long as a condition is True
  • D) Executes code after a delay
View Answer

Answer: C (A `while` loop continues until its condition evaluates to False)

5. Which keyword is used to exit a loop prematurely?

  • A) stop
  • B) exit
  • C) break
  • D) return
View Answer

Answer: C (`break` terminates the loop entirely)

6. Which keyword skips the rest of the current loop iteration and moves to the next one?

  • A) skip
  • B) continue
  • C) next
  • D) pass
View Answer

Answer: B (`continue` skips to the next iteration of the loop)

7. How do you loop 5 times using a `for` loop?

  • A) for i in 5:
  • B) for i = 1 to 5:
  • C) for i in range(5):
  • D) loop 5:
View Answer

Answer: C (`range(5)` generates numbers 0 through 4)

8. What numbers does `range(1, 4)` generate?

  • A) 1, 2, 3, 4
  • B) 1, 2, 3
  • C) 0, 1, 2, 3
  • D) 2, 3, 4
View Answer

Answer: B (`range(start, stop)` generates up to, but not including, the stop value)

9. What does the third argument in `range(1, 10, 2)` do?

  • A) It specifies the step size (increments by 2)
  • B) It specifies how many times to repeat
  • C) It specifies the default value
  • D) It causes an error
View Answer

Answer: A (It generates 1, 3, 5, 7, 9)

10. What usually causes an infinite `while` loop?

  • A) Using a `break` statement
  • B) Forgetting to update the loop condition variable
  • C) Using too much memory
  • D) Using `elif` inside the loop
View Answer

Answer: B (If the condition never becomes False, the loop runs forever)

11. Can you place an `if` statement inside another `if` statement?

  • A) Yes, this is called nested conditionals
  • B) No, it throws a SyntaxError
  • C) Only if they share the same variables
  • D) Only inside loops
View Answer

Answer: A (Nested conditionals are completely valid)

12. What is the `pass` statement used for?

  • A) To end a program
  • B) As a placeholder for future code (it does nothing)
  • C) To skip an iteration
  • D) To pass variables to a function
View Answer

Answer: B (`pass` avoids SyntaxErrors for empty blocks)

13. Will `if 5 > 2 and 10 < 20:` evaluate to True?

  • A) Yes
  • B) No
  • C) Throws an error
  • D) None
View Answer

Answer: A (Both conditions are True)

14. Will `if 5 > 10 or 10 < 20:` evaluate to True?

  • A) Yes
  • B) No
  • C) Throws an error
  • D) None
View Answer

Answer: A (Using `or` means only one side needs to be True)

15. Can a `for` loop have an `else` block in Python?

  • A) No, only `if` statements have `else`
  • B) Yes, and it executes if the loop finishes without hitting a `break`
  • C) Yes, but it executes on every iteration
  • D) Yes, but only if the loop throws an error
View Answer

Answer: B (The `else` block on a loop runs when the loop exhausts its iterable naturally)

16. Is the variable `i` from `for i in range(5):` accessible after the loop ends?

  • A) No, it is strictly scoped to the loop
  • B) Yes, it will retain the last value (4)
  • C) Yes, but its value will be None
  • D) It deletes itself
View Answer

Answer: B (Python does not have block-level scoping for loop variables)

17. How do you print multiple stars `*` on the same line in a loop?

  • A) print("*", end="")
  • B) print("*", no_newline=True)
  • C) printLine("*")
  • D) echo "*"
View Answer

Answer: A (The `end=""` argument replaces the default newline character)

18. Can you iterate over a string using a `for` loop?

  • A) No, only lists
  • B) Yes, `for char in "hello":` works
  • C) Only if you cast it to a list first
  • D) Yes, but it iterates by words, not characters
View Answer

Answer: B (Strings are iterables in Python)

19. What happens if you execute `if 0:`?

  • A) It evaluates to True
  • B) It evaluates to False
  • C) Syntax Error
  • D) Infinite loop
View Answer

Answer: B (`0` is a "Falsy" value in Python)

20. When would you use `while True:`?

  • A) To wait for a specific user input using `break` to exit
  • B) To freeze the program
  • C) To run a server indefinitely
  • D) Both A and C
View Answer

Answer: D (`while True` creates a deliberate infinite loop, often exited via `break` internally)
