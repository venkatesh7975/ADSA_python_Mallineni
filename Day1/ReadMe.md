## 🛠️ Tasks & Assignments

To solidify your understanding of today's topics, please complete the following tasks. Create a new Python file for each task (e.g., `task1.py`).

### Task 1: The Profile Builder (Variables & Data Types)
Create a script that stores information about a fictional employee.
1. Create variables for the employee's `first_name` (string), `last_name` (string), `age` (integer), `salary` (float), and `is_manager` (boolean).
2. Create a list called `skills` containing at least three strings (e.g., "Python", "Data Analysis", "SQL").
3. Print a formatted summary of the employee to the console using these variables.

### Task 2: The Temperature Converter (Arithmetic Operators)
Write a script to convert temperatures.
1. Declare a variable `temp_celsius` and assign it a numeric value (e.g., `25.5`).
2. Use arithmetic operators to calculate the Fahrenheit equivalent. The formula is: `(Celsius * 9/5) + 32`.
3. Store the result in a variable called `temp_fahrenheit`.
4. Print both temperatures clearly.

### Task 3: The Discount Checker (Logical & Comparison Operators)
Write a script that determines if a customer gets a discount.
1. Set a variable `cart_total` to `120.50`.
2. Set a variable `is_vip_member` to `True`.
3. A customer gets a 15% discount IF their cart total is over `$100` **OR** if they are a VIP member. 
4. Write a logical expression using `or`, `>`, and `==` to determine if they get the discount, and store the boolean result in a variable called `gets_discount`.
5. Print the result.

### Task 4: Fix the Errors (Debugging)
Copy the following code into a new script and fix all the errors so it runs successfully.
```python
1st_number = "10"
second_number = 5

# We want to add the two numbers mathematically
total = 1st_number + second_number

print("The total is: " total)
```
*(Hint: Watch out for invalid variable names, type conversion issues, and string concatenation rules!)*
