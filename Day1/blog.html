Variables & Data Types

The foundation of Python programming. Learn how to store, manipulate, and output data.

Today's Topics

  • Introduction to Python Syntax & Print Statement

  • Primitive Data Types (int, float, str, bool)
  • Basic Operators (Arithmetic, Relational, Logical)
  • Type Conversion (Casting)

Key Concepts

1. Variables

Variables act as containers for storing data values. In Python, you don't need to declare a variable's type explicitly.

name = "Alice"
age = 25
is_student = True

2. Basic Operators

Python supports standard mathematical operations, as well as string concatenation.

# Arithmetic
total = 10 + 5    # 15
power = 2 ** 3    # 8

# String Operations
greeting = "Hello " + name

Detailed Quiz (20 MCQs)

Test your knowledge on Variables, Data Types, and Operators!

1. What is the correct way to assign the value 10 to a variable named `x` in Python?

  • A) int x = 10;
  • B) x = 10
  • C) x := 10
  • D) declare x = 10
View Answer

Answer: B (Python is dynamically typed and uses `=` for assignment)

2. What data type is the value `10.5`?

  • A) int
  • B) float
  • C) str
  • D) bool
View Answer

Answer: B (Decimal numbers are represented as floats in Python)

3. How do you print "Hello World" to the console?

  • A) echo "Hello World"
  • B) p("Hello World")
  • C) print("Hello World")
  • D) console.log("Hello World")
View Answer

Answer: C (The `print()` function is used for output)

4. Which operator is used for exponentiation (power)?

  • A) ^
  • B) **
  • C) //
  • D) %
View Answer

Answer: B (`**` is the exponentiation operator)

5. What is the result of `10 % 3`?

  • A) 3.33
  • B) 3
  • C) 1
  • D) 0
View Answer

Answer: C (The modulo operator `%` returns the remainder. 10 divided by 3 leaves a remainder of 1)

6. What is the purpose of the `type()` function?

  • A) To convert a variable to a different type
  • B) To check the data type of a variable
  • C) To type input into the console
  • D) To create a new class
View Answer

Answer: B (`type(x)` returns the data type of `x`)

7. Which of the following is a valid string definition?

  • A) "Hello"
  • B) 'Hello'
  • C) """Hello"""
  • D) All of the above
View Answer

Answer: D (Python accepts single, double, and triple quotes for strings)

8. How are boolean values written in Python?

  • A) true / false
  • B) True / False
  • C) TRUE / FALSE
  • D) 1 / 0
View Answer

Answer: B (Booleans must be capitalized in Python)

9. Which of the following is an INVALID variable name?

  • A) my_var
  • B) _myvar
  • C) 2nd_var
  • D) var2
View Answer

Answer: C (Variable names cannot start with a number)

10. What does "dynamic typing" mean in Python?

  • A) Types can only be changed manually
  • B) You don't have to declare the type, and a variable can change types later
  • C) All variables are strings dynamically
  • D) Types are checked at compile time
View Answer

Answer: B (Variables take the type of the value assigned to them and can be reassigned to different types)

11. What does the `//` operator do?

  • A) Adds comments
  • B) Performs floating-point division
  • C) Performs floor division (removes decimal)
  • D) Finds the remainder
View Answer

Answer: C (Floor division rounds down to the nearest whole number)

12. What is the result of `True and False`?

  • A) True
  • B) False
  • C) None
  • D) Error
View Answer

Answer: B (The `and` operator requires both sides to be True)

13. What is the result of `not True`?

  • A) True
  • B) False
  • C) None
  • D) Error
View Answer

Answer: B (The `not` operator inverts the boolean value)

14. How do you concatenate two strings `a = "Hi"` and `b = "There"`?

  • A) a . b
  • B) a + b
  • C) a && b
  • D) concat(a, b)
View Answer

Answer: B (The `+` operator joins strings together)

15. In Python 3, what is the data type of the result of `5 / 2`?

  • A) int
  • B) float
  • C) str
  • D) decimal
View Answer

Answer: B (Standard division `/` always returns a float in Python 3)

16. What is the result of `int("5") + 5`?

  • A) "55"
  • B) 10
  • C) Error
  • D) "10"
View Answer

Answer: B (`int("5")` casts the string to an integer, making it 5 + 5 = 10)

17. What is the difference between `=` and `==`?

  • A) `=` compares values, `==` assigns values
  • B) `=` assigns values, `==` compares values for equality
  • C) They do the exact same thing
  • D) `==` is used for strict type checking
View Answer

Answer: B (`=` is assignment, `==` is a relational operator)

18. What is the result of `"5" + "5"`?

  • A) 10
  • B) "10"
  • C) "55"
  • D) Error
View Answer

Answer: C (Adding two strings performs concatenation)

19. How do you write a multi-line string in Python?

  • A) Using ///
  • B) Using ''' or """
  • C) Using ###
  • D) Using /* */
View Answer

Answer: B (Triple quotes allow strings to span multiple lines)

20. How do you write a single-line comment in Python?

  • A) // comment
  • B)
  • C) # comment
  • D) /* comment */
View Answer

Answer: C (The `#` symbol denotes a comment)
